## Star Shine

直播运营小工具合集

### Captain User

用户管理模块, 通过 xlsx 导入, 也可以直接在页面上添加用户

### Birthday Notice

生日提醒功能, 可以设置提醒时间, 发送微信通知, 发送桌面通知

### Revenue Manage

收益获取功能

模块一: 拉取特定日期范围的收益, 并且保存到 db 中;

模块二: 根据本地数据库中的数据进行保存, 和导出;

### 伴奏管理

- 基于爬虫的伴奏查询
